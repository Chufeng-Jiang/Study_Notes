    \textbf{Problem1-LeetcodeQ 217-Contains Duplicate-Easy}

Given an integer array nums, return true if any value appears at least
twice in the array, and return false if every element is distinct.

\begin{itemize}
\tightlist
\item
  Example 1:

  \begin{itemize}
  \tightlist
  \item
    Input: nums = {[}1,2,3,1{]}
  \item
    Output: true
  \end{itemize}
\item
  Example 2:

  \begin{itemize}
  \tightlist
  \item
    Input: nums = {[}1,2,3,4{]}
  \item
    Output: false
  \end{itemize}
\item
  Example 3:

  \begin{itemize}
